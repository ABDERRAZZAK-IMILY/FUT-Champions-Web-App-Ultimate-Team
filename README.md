## Description

**FUT Champions Ultimate Team** est une application web permettant aux joueurs de créer et gérer une équipe de football ultime dans le jeu **FIFA Ultimate Team**. Cette application permet de créer des joueurs, de les ajouter à une équipe, et de les déplacer entre différentes positions. Vous pouvez aussi gérer les remplaçants et personnaliser votre équipe selon vos préférences.

## Fonctionnalités

### Création de joueurs
- Créez des joueurs avec des caractéristiques personnalisées :
  - Nom
  - Note
  - Position
  - Vitesse
  - Tir
  - Passe
  - Dribble
  - Défense
  - Physique

### Gestion de l'équipe
- Ajoutez des joueurs à votre équipe principale et déplacez-les entre les différentes positions.
- Déplacez des joueurs entre l’équipe principale et les remplaçants.

### Interface utilisateur
- Interface intuitive et responsive, permettant une gestion fluide de votre équipe.

### API intégrée
- Utilisation d'une API pour récupérer des données sur les joueurs, ce qui vous permet d'ajouter facilement des joueurs à votre équipe avec leurs statistiques actuelles.

## Technologies utilisées

- **HTML** : Structure de la page web.
- **CSS** (Tailwind CSS) : Mise en page moderne et responsive.
- **JavaScript** : Logique pour l'ajout de joueurs, la gestion des positions et les interactions avec l'utilisateur.
- **API externe** : Utilisation d'une API pour récupérer les données des joueurs et les intégrer dans l'application.
